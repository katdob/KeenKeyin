# Welcome to KeenKeyin! 

Keen Key-in is the online presence for a password manager application. 

Full stack in TypeScript (React/Node.js). The goal is to set up basic profile information so I can make a basic application. This means there's going to be a button where you can sign up and sign in. Then and some API calls that send the data to/from the backend, and then the database stores everything necessary. 

I'm exploring Node.js and seeing what it can do since I haven't used this technology before. I'm not going to build a password manager quite yet. I may build the password manager in the future.
